Yeoman

Yeoman is a set of tools to make building web apps easier. Yeoman has support for Testacular via the Testacular Generator.

If you're also using AngularJS, check out the AngularJS Generator, which makes use of the Testacular Generator to setup a fully featured, testing-ready project.
